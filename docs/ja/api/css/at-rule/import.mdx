The `@import` CSS at-rule is used to import style rules from other stylesheets.

## Syntax

```css
@import <string>;
```

See [string](api/css/data-type/string.mdx) data type.

## Example

```css
@import 'custom.css';
@import './custom.css';
@import '/pages/card/index.css';
```
