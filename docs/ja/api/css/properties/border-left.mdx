# border-left

## Introduction

A shorthand property for setting the left border styles of an element. It includes [`border-left-width`](./border-left-width), [`border-left-style`](./border-left-style), and [`border-left-color`](./border-left-color) in order.

## Syntax

```css
border-left: 1px solid red;
```

## Values

Can be set in order as follows:

- [`border-left-width`](./border-left-width.mdx)
  The width of the border.

- [`border-left-style`](./border-left-style.mdx)
  The style of the border.

- [`border-left-color`](./border-left-color.mdx)
  The color of the border.


## Differences from the Web

- [mdn:border-left](https://developer.mozilla.org/en-US/docs/Web/CSS/border-left)
- Does not support `inherit`, `initial`, `revert`, or `unset`.
