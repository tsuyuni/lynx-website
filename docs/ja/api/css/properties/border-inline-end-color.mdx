## Introduction

The `border-inline-end-color` CSS property defines the color of the logical inline-end border of an element.

In a horizontal-tb writing mode with `LTR` direction, it corresponds to the [`border-right-color`](./border-right-color) property.
In a horizontal-tb writing mode with `RTL` direction, it corresponds to the [`border-left-color`](./border-left-color) property.

## Syntax

```css
border-inline-end-color: red;
border-inline-end-color: #ffbb00;
border-inline-end-color: rgb(255, 0, 0);
border-inline-end-color: hsla(100%, 50%, 25%, 0.75);
border-inline-end-color: currentColor;
border-inline-end-color: transparent;
```

### Values

- `transparent`

- [`<color>`](api/css/data-type/color)
