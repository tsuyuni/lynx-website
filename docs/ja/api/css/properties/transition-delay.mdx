# transition-delay

## Introduction

The transition-delay CSS property specifies the duration to wait before starting a property's transition effect when its value changes.

## Syntax

```css
transition-delay: 2s;
transition-delay: 300ms;
```

### Values

- [`<time>`](api/css/data-type/time.mdx)

- Default Value
  **0s**
