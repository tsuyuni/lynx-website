## Introduction

The `row-gap`/`grid-row-gap` CSS property sets the size of the gap (gutter) between an element's rows in [flexible box layout](guide/ui/layout/flexible-box-layout.mdx) and grid layout.

For compatibility, `row-gap` or `grid-row-gap` can be used both.

## Values

### `<length>`

[`<length>`](api/css/data-type/length.mdx) is the width of the gutter separating the rows.

### `<percentage>`

[`<percentage>`](api/css/data-type/percentage.mdx) values are relative to the dimension of the element.

## Formal syntax

## Syntax

```css
/* <length> values */
row-gap: 20px;
row-gap: 1em;

/* <percentage> value */
row-gap: 10%;

/* calc() values */
row-gap: calc(10% + 20px);
```
