# -x-handle-color

## Introduction

The `-x-handle-color` property specifies the color of the floating marker when copying text, only valid in the selection pseudo-element.

## Syntax

```css
::selection {
  -x-handle-color: blue;
}
```

### Values

- [`<color>`](./color.mdx)<br/>
  Sets the color of the floating marker
