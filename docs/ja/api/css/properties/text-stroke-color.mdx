## Introduction

`text-stroke-color` specifies the stroke color of characters of text.

## Syntax

```css
text-stroke-color: red;
text-stroke-color: #e08ab4;
text-stroke-color: rgb(200, 100, 0);
```

### Values

- [`<color>`](./color.mdx)<br/>
  The color of the stroke.
