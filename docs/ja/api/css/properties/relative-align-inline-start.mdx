## Introduction

Specifies that the current element is aligned with the left/right edges of the parent or sibling element corresponding to [`id`](./relative-id.mdx).

When [`direction`](./direction.mdx) is `ltr`, `relative-align-inline-start` is equivalent to [`relative-align-left`](./relative-align-left.mdx);

When [`direction`](./direction.mdx) is `rtl/lynx-rtl`,`relative-align-inline-start` is equivalent to [`relative-align-right`](./relative-align-right.mdx);

## Syntax

```css
relative-align-inline-start: parent;
relative-align-inline-start: none;

relative-align-inline-start: 1;
relative-align-inline-start: 2;
```

### Values

The `relative-align-inline-start` property takes the same values as the [`relative-align-left`](./relative-align-left) property.
