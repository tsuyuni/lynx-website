## Introduction

The `border-inline-start-color` CSS property defines the color of the logical inline start border of an element.

In a horizontal-tb writing mode with `LTR` direction, it corresponds to the [`border-left-color`](./border-left-color) property.
In a horizontal-tb writing mode with `RTL` direction, it corresponds to the [`border-right-color`](./border-right-color) property.

## Syntax

```css
border-inline-start-color: red;
border-inline-start-color: #ffbb00;
border-inline-start-color: rgb(255, 0, 0);
border-inline-start-color: hsla(100%, 50%, 25%, 0.75);
border-inline-start-color: currentColor;
border-inline-start-color: transparent;
```
