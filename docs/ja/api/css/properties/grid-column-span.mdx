# <Deprecated /> grid-column-span

:::warning
This API is deprecated.
:::

## Introduction

Contributes a grid span to the grid item's placement.
If there are [`grid-column-start`](./grid-column-start) and [`grid-column-end`](./grid-column-end), this css property will be overridden.

## Syntax

```css
/* <number> values */
grid-column-span: 1;
grid-column-span: 2;
```

### Values

- `<number>`

  Contributes a grid span to the grid item's placement.
