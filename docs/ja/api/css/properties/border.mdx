## Introduction

The border shorthand CSS property sets an element's border. It sets the values of [border-width](./border-width.mdx), [border-style](./border-style.mdx), and [border-color](./border-color.mdx).

## Syntax

```css
/* style */
border: solid;

/* width | style */
border: 2px dotted;

/* style | color */
border: outset #f33;

/* width | style | color */
border: medium dashed green;
```

### Values

This property is a shorthand for the following CSS properties:

- [`border-width`](./border-width.mdx)
  Sets the thickness of the border.

- [`border-style`](./border-style.mdx)
  Sets the style of the border.

- [`border-color`](./border-color.mdx)
  Sets the color of the border.

## Formal syntax

```
<line-width> || <line-style> || <color>
where
<line-width> = <length> | thin | medium | thick
<line-style> = none | hidden | dotted | dashed | solid | double | groove | ridge | inset | outset

```

## Difference between web

- [mdn:border](https://developer.mozilla.org/en-US/docs/Web/CSS/border)
