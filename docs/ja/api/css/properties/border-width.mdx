# border-width

## Introduction

A shorthand property for setting the width of all four borders of an element. It corresponds to [`border-top-width`](./border-top-width.mdx), [`border-right-width`](./border-right-width.mdx), [`border-bottom-width`](./border-bottom-width.mdx), and [`border-left-width`](./border-left-width.mdx).

## Syntax

```css
border-width: thin medium thick 10px;
```

### Values

- thin | medium | thick | [`<length>`](/api/css/data-type/length.html)
- Supports 1-4 value syntax (top → right → bottom → left).

## Formal Syntax

```css
border-width = [ <line-width> ]{1,4}
<line-width> = <length> | thin | medium | thick
```

## Differences from the Web

- [MDN Reference](https://developer.mozilla.org/en-US/docs/Web/CSS/border-width)
- Default value is different (Web default is `medium`).
- Does not support global values.
