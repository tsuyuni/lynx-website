## Introduction

The `inset-inline-end` CSS property defines the logical inline end inset of an element, which maps to a physical offset depending on the element's directionality. It corresponds to the [`right`](./right) or [`left`](./left) property depending on the values defined for [`direction`](./direction).

In `LTR`, it corresponds to the [`right`](./right) property.
In `RTL`, it corresponds to the [`left`](./left) property.

## Syntax

```css
/* <length> values */
inset-inline-end: 3px;
inset-inline-end: 2rpx;
inset-inline-end: 2.4em;
inset-inline-end: 3rem;

inset-inline-end: 10%;

/* Keyword value */
inset-inline-end: auto;

/* calc */
inset-inline-end: calc(1px + 1px);
```

### Values

The `inset-inline-end` property takes the same values as the [`left`](./left) property.
