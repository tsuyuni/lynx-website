# PerformanceObserver

The `PerformanceObserver` is used to observe performance events and receive notifications when [Lynx Engine](guide/spec#engine) generates [performance entries](api/lynx-api/performance-api/performance-entry).
It must be created through [`lynx.performance.createObserver()`](api/lynx-api/lynx/lynx-performance#createobserver) and bind with a callback function for handling received performance entries upon creation.

## Instance methods

[`PerformanceObserver.observe()`](./performance-observer/observe)

Specify a set of performance entry types to observe and start observing.

[`PerformanceObserver.disconnect()`](./performance-observer/disconnect)

Stop receiving performance entries.
