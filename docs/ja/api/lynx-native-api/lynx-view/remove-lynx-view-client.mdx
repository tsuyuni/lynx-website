# removeLynxViewClient

remove `LynxView` lifecycle event monitoring;

## Syntax

### Android

```java
public void removeLynxViewClient(LynxViewClient client)
```

## Parameters

- `client: LynxViewClient`: The structure implemented by the client and registered to the LynxView instance is used to obtain the callbacks of each process in the LynxView life cycle.
