# UpdateData

Using [LynxUpdateMeta](/api/lynx-native-api/lynx-update-meta) to update `LynxView`, it is the main entrance for the client to update template data.

## Syntax

### Android

```java
void updateData(LynxUpdateMeta meta)
```

### Parameters

- `meta: LynxUpdateMeta`: Lynx template update metadata, developers use this data structure to specify the optional data content of the update template;

### iOS

```objc
(void)updateData:(nonnull LynxUpdateMeta*)meta;
```

### Parameters

- `meta: LynxUpdateMeta*`: Lynx template update metadata, developers use this data structure to specify the optional data content of the update template;
