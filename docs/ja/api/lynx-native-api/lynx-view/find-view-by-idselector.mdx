# findViewByIdSelector

Find `UIView` based on `idSelector` property

## Syntax

### Android

```java
public LynxBaseUI findUIByIdSelector(String id);
```

### Parameters

- `id: String`: The id attribute of the view specified by the front end in the layout file;

### iOS

```objc
- (nullable LynxUI*)uiWithIdSelector:(nonnull NSString*)idSelector;
```

## Parameters

- `idSelector: NSString*`: The id attribute of the view specified by the front end in the layout file;
