# findViewByIdName

Find `UIView` based on `name` attribute

## Syntax

### Android

```java
public View findViewByName(String name);
```

### Parameters

- `name: String`: The name attribute of the view specified by the front end in the layout file;

### iOS

```objc
- (nullable UIView*)viewWithName:(nonnull NSString*)name;
```

### Parameters

- `name: NSString*`: The name attribute of the view specified by the front end in the layout file;
