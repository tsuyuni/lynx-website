# クイックスタート

Lynx のドキュメントへようこそ! Lynx プロジェクトを作成し、開発を始めましょう。

## システム要求

- [Node.js 18](https://nodejs.org/en) 以降
  - TypeScript を使用する場合は Node.js 18.19 が必要です。

## インストール

<Steps>

### Lynx プロジェクトを作成する

Lynx プロジェクトの構築には Rspeedy（Rspack ベースの Lynx ビルドツール）を使用します。

[`create-rspeedy`](https://npmjs.org/package/create-rspeedy)を用いてプロジェクトを開始することで、自動セットアップが可能です。プロジェクトを作成するには、以下のコマンドを実行します。

<PackageManagerTabs command="create rspeedy@latest" />

このコマンドの後、 `create-rspeedy` でプロジェクト名のフォルダが作成されます。

### Lynx Explorer を用意する

Lynx Explorer は Lynx を素早く試すためのサンドボックス環境です。

<PlatformTabs hashKey="explorer-platform">

<PlatformTabs.Tab platform="ios-simulator">

:::info
現在、iOS シミュレータ用のプリビルドバイナリのみ提供しています。実際の iOS デバイスで Lynx Explorer を実行する場合は、ソースからビルドする必要があります。[Build Lynx Explorer for iOS](https://github.com/lynx-family/lynx/tree/develop/explorer/darwin/ios)を参照してください。
:::

1. **Xcode をインストール**

Mac App Store を開いて[Xcode](https://apps.apple.com/us/app/xcode/id497799835)を検索し、インストール（または更新）をクリックします。

2. **<div id="download-lynx-explorer,ios-simulator-platform=macos-arm64,explorer-platform=ios-simulator">LynxExplorer をダウンロード</div>**

<PlatformTabs hashKey="ios-simulator-platform">

<PlatformTabs.Tab platform="macos-arm64">

[`LynxExplorer-arm64.app.tar.gz`](https://github.com/lynx-family/lynx/releases/latest/download/LynxExplorer-arm64.app.tar.gz)をダウンロードします。

続いて、ダウンロードしたアーカイブを展開します:

```bash
mkdir -p LynxExplorer-arm64.app/
tar -zxf LynxExplorer-arm64.app.tar.gz -C LynxExplorer-arm64.app/
```

3. **LynxExplorer をシミュレータにインストール**

XCode を開き、Xcode メニューから **Open Developer Tool** を選択します。**Simulator** をクリックして起動し、"LynxExplorer-arm64.app" をドラッグアンドドロップします。

</PlatformTabs.Tab>

<PlatformTabs.Tab platform="macos-intel">

[`LynxExplorer-x86_64.app.tar.gz`](https://github.com/lynx-family/lynx/releases/latest/download/LynxExplorer-x86_64.app.tar.gz)をダウンロードします。

続いて、ダウンロードしたアーカイブを展開します:

```bash
mkdir -p LynxExplorer-x86_64.app/
tar -zxf LynxExplorer-x86_64.app.tar.gz -C LynxExplorer-x86_64.app/
```

3. **LynxExplorer をシミュレータにインストール**

XCode を開き、Xcode メニューから **Open Developer Tool** を選択します。**Simulator** をクリックして起動し、"LynxExplorer-x86_64.app" をドラッグアンドドロップします。

</PlatformTabs.Tab>

</PlatformTabs>

</PlatformTabs.Tab>

<PlatformTabs.Tab platform="android">

QR コードをスキャンして、[GitHub Release](https://github.com/lynx-family/lynx/releases/latest)からプリビルドアプリをダウンロードしてください。

<QRCodeSVG

[Build Lynx Explorer for Android](https://github.com/lynx-family/lynx/tree/develop/explorer/android)に従ってソースからビルドすることもできます。

</PlatformTabs.Tab>

</PlatformTabs>

### 開発を始める

1. プロジェクトのディレクトリに移動:

```bash
cd <project-name>
```

2. パッケージマネージャを用いて NPM パッケージの依存関係をインストール:

<PackageManagerTabs command="install" />

3. 開発用サーバを起動:

<PackageManagerTabs command="run dev" />

ターミナルに表示される QR コードを Lynx Explorer アプリでスキャンしてください。

4. コードに変更を加える

コードエディタで `src/App.tsx` ファイルを開いて変更を加えてください。

Lynx Explorer 上で UI が自動的に更新されるのが確認できます。

<Go
  example="hello-world"
  defaultFile="src/App.tsx"
  img="https://lf-lynx.tiktok-cdns.com/obj/lynx-artifacts-oss-sg/lynx-website/assets/doc/hello-world-showcase-ios.png"
  defaultEntryFile="dist/main.lynx.bundle"
/>

<br />

### デバッグを行う

[Lynx DevTool](https://github.com/lynx-family/lynx-devtool/releases)から Lynx DevTool デスクトップアプリケーションをダウンロードして開きます。デバッグ対象のデバイスを USB ケーブルで接続し、デバッグを開始します。

[Debugging](/guide/debugging/lynx-devtool.html)を参照して、Lynx アプリのデバッグ方法を学びます。

</Steps>
