# 既存のアプリケーションと統合する

現在、Lynx は新しいアプリケーションをゼロから構築するためには適していません。Lynx（エンジン）をネイティブモバイルアプリケーションまたは Web アプリケーションに統合し、Lynx ビューを介して Lynx アプリケーションをロードする必要があります。Lynx でのアプリケーション開発は数ステップで開始できます。

統合のためのステップを確認するためには、対象プラットフォームを選択してください：
