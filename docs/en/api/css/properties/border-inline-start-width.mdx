## Introduction

The `border-inline-start-width` CSS property defines the width of the logical inline-start border of an element.

In a horizontal-tb writing mode with `LTR` direction, it corresponds to the [`border-left-width`](./border-left-width) property.
In a horizontal-tb writing mode with `RTL` direction, it corresponds to the [`border-right-width`](./border-right-width) property.

## Syntax

```css
border-inline-start-width: thin;
border-inline-start-width: medium;
border-inline-start-width: thick;
border-inline-start-width: 10px;
```

### Values

- `thin`

- `medium`

- `thick`

- [`<length>`](api/css/data-type/length)
