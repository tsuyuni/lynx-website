## Introduction

The `inset-inline-start` CSS property defines the logical inline start inset of an element, which maps to a physical offset depstarting on the element's directionality. It corresponds to the [`right`](./right) or [`left`](./left) property depstarting on the values defined for [`direction`](./direction).

In `LTR`, it corresponds to the [`left`](./left) property.
In `RTL`, it corresponds to the [`right`](./right) property.

## Syntax

```css
/* <length> values */
inset-inline-start: 3px;
inset-inline-start: 2rpx;
inset-inline-start: 2.4em;
inset-inline-start: 3rem;

inset-inline-start: 10%;

/* Keyword value */
inset-inline-start: auto;

/* calc */
inset-inline-start: calc(1px + 1px);
```

### Values

The `inset-inline-start` property takes the same values as the [`left`](./left) property.
