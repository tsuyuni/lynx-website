## Introduction

The current element is to the left/right of the sibling element corresponding to the specified [`id`](./relative-id.mdx) in [relative-layout](guide/ui/layout/relative-layout.mdx).

When [`direction`](./direction.mdx) is `ltr`, `relative-inline-end-of` is equivalent to [`relative-right-of`](./relative-right-of.mdx);

When [`direction`](./direction.mdx) is `rtl/lynx-rtl`, `relative-inline-end-of` is equivalent to [`relative-left-of`](./relative-left-of.mdx).

## Values

The `relative-inline-end-of` property takes the same values as the [`relative-left-of`](./relative-left-of.mdx).

## Syntax

```css
relative-inline-end-of: none;
relative-inline-end-of: 1;
relative-inline-end-of: 2;
```
