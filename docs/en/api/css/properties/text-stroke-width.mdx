## Introduction

The `text-stroke-width` property specifies the width of the text stroke.

## Syntax

```css
text-stroke-width: 2px;
text-stroke-width: 0.1em;
text-stroke-width: 5pt;
```

### Values

- [`<length>`](api/css/data-type/length.mdx)<br/>
  The width of the stroke.
