# InitBackgroundRuntimeEntry

`InitBackgroundRuntimeEntry` 接口提供关于 [Background Thread Runtime](guide/spec#background-thread-runtime) 初始化的关键时间，它继承自 [`PerformanceEntry`](api/lynx-api/performance-api/performance-entry)。

## 实例属性

### [entryType](api/lynx-api/performance-api/performance-entry#entryType)

```tsx
entryType: string;
```

性能事件的类型，当前类所有实例的值均固定为 `init`。

### [name](api/lynx-api/performance-api/performance-entry#name)

```tsx
name: string;
```

性能事件的名称，当前类所有实例的值均固定为 `backgroundRuntime`。

### loadCoreStart

```tsx
loadCoreStart: number;
```

初始化[后台线程运行时](guide/spec#background-thread-runtime)环境的开始时间戳。该时间戳是一个表示为浮点型的 Unix 时间戳（单位：毫秒），精确到小数点后三位。例如：1739594612307.429。

### loadCoreEnd

```tsx
loadCoreEnd: number;
```

初始化[后台线程运行时](guide/spec#background-thread-runtime)环境的结束时间戳。该时间戳是一个表示为浮点型的 Unix 时间戳（单位：毫秒），精确到小数点后三位。例如：1739594612307.429。
