# padding-bottom

## 介绍

指定一个元素内容与下边界的距离，该属性不能为负值。

## 语法

```css
/* <length> values */
padding-bottom: 0.5em;
padding-bottom: 0;

/* <percentage> value */
padding-bottom: 10%;
```

### 取值

#### `0px`

**默认值**。

#### `<length>`

[`<length>`](api/css/data-type/length.mdx) 以固定值为内边距。不能为负值。

#### `<percentage>`

相对于[包含块](https://developer.mozilla.org/zh-CN/docs/Web/CSS/Containing_block)的行内尺寸（宽度）的百分比。必须是非负值。
