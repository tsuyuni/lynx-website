# padding-inline-start

## 介绍

定义了元素的逻辑行首内边距，并根据元素的 [`direction`](./direction) 对应至实体内边距。该属性不能为负值。

在 [`LTR`] 模式下，`padding-inline-start` 等价于 [`padding-left`](./padding-left)。
在 `RTL` 模式下，`padding-inline-start` 等价于 [`padding-right`](./padding-right)。

## 语法

```css
/* <length> values */
padding-inline-start: 0.5em;
padding-inline-start: 0px;
padding-inline-start: 2rpx;

/* <percentage> value */
padding-inline-start: 10%;
```

### 取值

#### `0px`

**默认值**。

#### `<length>`

[`<length>`](api/css/data-type/length.mdx) 定义为一个绝对值。不能为负值。

#### `<percentage>`

相对于[包含块](https://developer.mozilla.org/zh-CN/docs/Web/CSS/Containing_block)的行内尺寸（宽度）的百分比。必须是非负值。
