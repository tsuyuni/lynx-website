# max-height

## 介绍

设置元素的最大高度。它可以阻止 [`height`](./height.mdx) 属性的应用值大于 `max-height` 的值。
`max-height` 的值会覆盖 [`height`](./height.mdx) 和 [`min-height`](./min-height.mdx)。

## 语法

```css
/* <length> */
max-height: 120px;
max-height: 10em;

/* <percentage> */
max-height: 75%;
```

### 取值

#### 默认盒子高度没有限制

#### `<length>`

[`<length>`](api/css/data-type/length.mdx) 定义了一个确定值。

#### `<percentage>`

定义作为包含区块高度百分比的 `max-height`。

## 形式语法

```
max-height = <length-percentage [0,∞]>

<length-percentage> =
  <length>      |
  <percentage>
```

## 与 Web 的区别

[mdn: max-height](https://developer.mozilla.org/en-US/docs/Web/CSS/max-height)

1. 不支持 `none`，`max-content`，`min-content`，`fit-content`。
2. 在 Lynx 中 `max-height` 会覆盖 [`min-height`](./min-height.mdx)。
