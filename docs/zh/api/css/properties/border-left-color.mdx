# border-left-color

## 介绍

`border-left-color` 属性设置元件的左边框颜色。值得注意的是，在大多数情况下使用 [`border-color`](./border-color.mdx) 或 [`border-left`](./border-left.mdx) 更加方便和普遍。

## 语法

```css
border-left-color: red;
border-left-color: #ffbb00;
border-left-color: rgb(255, 0, 0);
border-left-color: hsla(100%, 50%, 25%, 0.75);
```

## 与 Web 的区别

- 不支持 `Global values` 例如：`inherit`，`initial`，`unset`
