# border-style

## 介绍

用于设置四个边框的样式，该属性可按顺序设置上边框样式、右边框样式、下边框样式、左边框样式。

## 语法

```css
border-style: double solid dotted dashed;
```

### 取值

- none | hidden | dotted | dashed | solid | double | groove | ridge | inset | outset
- 支持 1 - 4 值语法（上 → 右 → 下 → 左）

## 形式语法

```css

border-style = [ <line-style> ]{1,4}
<line-style> = none | hidden | dotted | dashed | solid | double | groove | ridge | inset | outset

```

## 与 Web 的区别

- [MDN 参考](https://developer.mozilla.org/zh-CN/docs/Web/CSS/border-style)
- 默认值不同（Web 为 `none`）
- 值为 `none`/`hidden` 时边框宽度保持原值（Web 会置为 `0`）
- 不支持 `inherit`/`initial`/`revert`/`unset` 全局值
