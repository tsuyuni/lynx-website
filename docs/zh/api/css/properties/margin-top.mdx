# margin-top

## 介绍

设置元件的顶部外边距外边距区域。正值使它离相邻元件更远，而负值使它更靠近相邻元件。

## 语法

```css
/* <length> values */
margin-top: 0.5em;
margin-top: 0;
margin-top: 2rpx;

/* <percentage> value */
margin-top: 10%;
```

### 取值

#### `0px`

**默认值**。

#### `<length>`

[`<length>`](api/css/data-type/length.mdx) 定义了一个确定的宽度值。可以为负值。

#### `<percentage>`

相对于[包含块](https://developer.mozilla.org/zh-CN/docs/Web/CSS/Containing_block)的宽度，以百分比值为外边距。

#### `auto`

让布局流程自己选择一个合适的外边距。

:::info
目前支持在[弹性布局](https://developer.mozilla.org/zh/docs/Web/CSS/CSS_flexible_box_layout)、[网格布局](https://developer.mozilla.org/zh/docs/Web/CSS/CSS_grid_layout)、[线性布局](guide/ui/layout/linear-layout.mdx)的非 [`position: fixed/absolute/sticky`](./position) 子元件中使用。

其中，对于线性布局，仅支持在交叉轴方向上使用 `auto`。
:::

## 形式语法

```
margin-top =
  <length-percentage>  |
  auto                 |

<length-percentage> =
  <length>      |
  <percentage>
```

## 与 Web 的区别

[mdn: margin-top](https://developer.mozilla.org/zh-CN/docs/Web/CSS/margin-top)

1. 在 Lynx 下，`margin` 任何情况都不会重叠。
2. 目前支持在[弹性布局](https://developer.mozilla.org/zh/docs/Web/CSS/CSS_flexible_box_layout)、[网格布局](https://developer.mozilla.org/zh/docs/Web/CSS/CSS_grid_layout)、[线性布局](guide/ui/layout/linear-layout.mdx)的非 [`position: fixed/absolute/sticky`](./position) 子元件中使用 `auto`。其中，对于线性布局，仅支持在交叉轴方向上的外边距 `auto` 值。
