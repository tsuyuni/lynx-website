# border-bottom-color

## 介绍

设置元素下边框颜色。需单独设置下边框颜色时使用。

## 语法

```css
border-bottom-color: red;

border-bottom-color: '#ff0000';
```

### 取值

- [`<color>`](/api/css/data-type/color.mdx)

## 形式语法

```css

border-bottom-color = <color>
<color> = <rgb()> | <rgba()> | <hsl()> | <hsla()> | <hex-color> | <named-color>

```

## 与 Web 的区别

- [MDN 参考](https://developer.mozilla.org/zh-CN/docs/Web/CSS/border-bottom-color)
- 默认值不同（Web 为 `currentColor`）
- 不支持 inherit、initial、revert、unset 等全局值
