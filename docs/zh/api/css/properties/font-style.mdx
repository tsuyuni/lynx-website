## 介绍

设置字体样式。

## 语法

```css
font-style: italic;
font-style: oblique;
font-style: normal;
```

### 取值

- **`normal`**
  普通

- `italic`
  斜体字

- `oblique`
  倾斜

## 形式定义

import { PropertyDefinition } from '@/components/PropertyDefinition';

<PropertyDefinition
  initialValue={<>normal</>}

/>

## 形式语法

```
font-style = normal | italic | oblique
```

## 与 Web 的区别

- 只支持 `normal`、`oblique` 和 `italic`

- [mdn:font-style](https://developer.mozilla.org/en-US/docs/Web/CSS/font-style)
