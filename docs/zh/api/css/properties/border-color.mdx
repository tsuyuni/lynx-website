# border-color

## 介绍

`border-color` 属性是一个用于设置元件四个边框颜色 ([`border-top-color`](./border-top-color.mdx), [`border-right-color`](./border-right-color.mdx), [`border-bottom-color`](./border-bottom-color.mdx), [`border-left-color`](./border-left-color.mdx)) 的快捷属性。

## 语法

```css
/* border-color: color; 单值语法 */
border-color: red;

/* border-color: vertical horizontal; 双值语法*/
border-color: red #f015ca;

/* border-color: top horizontal bottom; 三值语法 */
border-color: red yellow green;

/* border-color: top right bottom left; 四值语法 */
border-color: red yellow green blue;
```

### 取值

- [`<color>`](api/css/data-type/color.mdx) 支持单值、双值、三值和四值语法。 <br/>
  - 单值语法：<br/>
    同时指定四个边框的颜色。
  - 双值语法：<br/>
    第一个值指定上边框 [`border-top-color`](./border-top-color.mdx) 和下边框 [`border-bottom-color`](./border-bottom-color.mdx) 的值。<br/>
    第二个值指定左边框 [`border-left-color`](./border-left-color.mdx) 和右边框 [`border-right-color`](./border-right-color.mdx) 的值。
  - 三值语法：<br/>
    第一个值指定上边框 [`border-top-color`](./border-top-color.mdx) 的值。<br/>
    第二个值指定左边框 [`border-left-color`](./border-left-color.mdx) 和右边框 [`border-right-color`](./border-right-color.mdx) 的值。<br/>
    第三个值指定下边框 [`border-bottom-color`](./border-bottom-color.mdx) 的值。
  - 四值语法：<br/>
    第一个值指定上边框 [`border-top-color`](./border-top-color.mdx) 的值。<br/>
    第二个值指定右边框 [`border-right-color`](./border-right-color.mdx) 的值。<br/>
    第三个值指定下边框 [`border-bottom-color`](./border-bottom-color.mdx) 的值。<br/>
    第四个值指定左边框 [`border-left-color`](./border-left-color.mdx) 的值。
