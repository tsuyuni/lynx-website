# relative-inline-start-of

## 介绍

[相对布局](guide/ui/layout/relative-layout.mdx)中，指定当前元素在指定对应 [`id`](./relative-id.mdx) 的同级元素的左/右侧。

当 [`direction`](./direction.mdx) 为 `ltr` 时，`relative-inline-start-of` 等价于 [`relative-left-of`](./relative-left-of.mdx)；

当 [`direction`](./direction.mdx) 为 `rtl/lynx-rtl` 时，`relative-inline-start-of` 等价于 [`relative-right-of`](./relative-right-of.mdx)。

## 语法

```css
relative-inline-start-of: none;
relative-inline-start-of: 1;
relative-inline-start-of: 2;
```

### 取值

取值与 [`relative-left-of`](./relative-left-of) 属性相同。

```

```

## 与 Web 的区别

- Web 无此属性。
