# justify-items

## 介绍

在[网格布局](guide/ui/layout/grid-layout.mdx)中，设置网格区域（grid area）中的元素在**行轴**方向的轴上对齐。此属性作用在网格容器上。[`justify-self`](./justify-self) 可覆盖网格容器定义的 `justify-items`。

:::info
Lynx 不支持 `writing-mode`，所以行轴就是水平轴。
:::

## 取值

### `stretch`

**默认值**。除非元素具有确定的宽度。也就是说在默认情况下，元素会在行轴方向上覆盖网格区域。

### `center`

定义元素在网格区域的行轴方向上居中。

### `start`

定义元素在行轴上与网格区域的起始边缘对齐。

### `end`

定义元素在行轴上与网格区域的结束边缘对齐。

## 语法

```css
justify-items: stretch;
justify-items: center;
justify-items: start;
justify-items: end;
```

## 形式语法

```
justify-items = stretch | center | start | end
```

## 与 Web 的区别

[mdn: justify-items](https://developer.mozilla.org/zh-CN/docs/Web/CSS/justify-items)

1. 不支持 `normal`。默认值为 `stretch`。
2. 不支持 `flex-start`、`flex-end`、`self-start`、`self-end`、`left`、`right`。
