# max-width

## 介绍

设置元件的最大宽度。它可以阻止 [`width`](./width.mdx) 属性的应用值大于 `max-width` 的值。
`max-width` 的值会覆盖 [`width`](./width.mdx) 和 [`min-width`](./min-width.mdx)。

## 语法

```css
/* <length> */
max-width: 120px;
max-width: 10em;

/* <percentage> */
max-width: 75%;
```

### 取值

#### 默认盒子宽度没有限制

#### `<length>`

[`<length>`](api/css/data-type/length.mdx) 定义了一个确定值。

#### `<percentage>`

以包含区块的宽度百分比定义 `max-width`。

## 形式定义

## 形式语法

```
max-width = <length-percentage [0,∞]>

<length-percentage> =
  <length>      |
  <percentage>
```

## 与 Web 的区别

[mdn: max-width](https://developer.mozilla.org/en-US/docs/Web/CSS/max-width)

1. 不支持 `none`，`max-content`，`min-content`，`fit-content`。
2. 在 Lynx 中 `max-width` 会覆盖 [`min-width`](./min-width.mdx)。
