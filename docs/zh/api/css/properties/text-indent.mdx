# text-indent

## 介绍

text-indent 属性能定义一个 text 首行文本内容之前的缩进量

## 使用示例

import { Go } from '@lynx';

<Go
  example="css-api"

## 语法

```css
/* <length> values */
text-indent: 40px;

/* <percentage> value
   relative to the containing block width */
text-indent: 15%;
```

### 取值

- [`<length>`](api/css/data-type/length.mdx)
  长度，默认值为 0

- [`<percentage>`](api/css/data-type/percentage.mdx)
  参照于 text 宽度的百分比

## 形式定义

import { PropertyDefinition } from '@/components/PropertyDefinition';

## 形式语法

```
<length> | <percentage>
```

## 与 Web 的区别

在写**百分比**的时候需要声明 `text` 元件的宽度，否则会不生效
