# relative-bottom-of

## 介绍

[相对布局](guide/ui/layout/relative-layout.mdx)中，设置当前元素在指定对应 [`id`](./relative-id.mdx) 的同级元素的下方。

## 语法

```css
relative-bottom-of: none;
relative-bottom-of: 1;
relative-bottom-of: 2;
```

### 取值

- `none`

  **默认值**。不设置此属性。

- `<number>`

  `int` 值。设置当前元素在指定对应 [`id`](./relative-id.mdx) 的同级元素的下方。

  :::info
  [`id`](./relative-id.mdx) 的设定请注意：`-1` 和 `0` 是特殊值。`-1` 是默认值，代表当前元素不会被视为同级元素的对齐参考标准。不能使用 `0` 作为元素的 id，否则有不符合预期的表现。
  :::

```

```

## 与 Web 的区别

- Web 无此属性。
