# line-height

## 介绍

`line-height` 属性用于设置多行文本的间距。

## 语法

```css
line-height: normal;

/* Unitless values: use this number multiplied
by the element's font size */
line-height: 3.5;
```

### 取值

- normal <br/>
  默认值。

- [`<number>`](api/css/data-type/number.mdx) <br/>
  无单位的数字，表示行高为该数字乘以该元件的字体大小 ([`font-size`](./font-size.mdx))。

- [`<length>`](api/css/data-type/length.mdx) <br/>
  指定行高的具体数值。

```
normal | <length> | <number>
```

## 与 Web 的区别

- `line-height` 属性只影响 `<text>` 元件，设置在其他标签上不会生效。

## 注意事项

- `line-height` 的值最好大于 `font-size` 的值，否则文本上下可能会被截断。

- `line-height` 属于段落属性，不会作用于 inline text 上。
