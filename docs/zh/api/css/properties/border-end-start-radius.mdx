# border-end-start-radius

## 介绍

用于添加阅读模式开始方向下角圆角边框。
第一个值是水平半径，第二个值是垂直半径。如果省略第二个值，则复制第一个值。如果长度为零，则边角为方形，而不是圆形。水平半径的百分比值参考边框盒的宽度，而垂直半径的百分比值参考边框盒的高度。

在 `LTR` 模式下，`border-end-start-radius` 等价于 `border-bottom-left-radius`。
在 `RTL` 模式下，`border-end-start-radius` 等价于 `border-bottom-right-radius`。

## 语法

```css
/* the corner is a circle */
/* border-end-start-radius: radius */
border-end-start-radius: 3px;

/* the corner is an ellipsis */
/* border-end-start-radius: horizontal vertical */
border-end-start-radius: 0.5em 1em;
```

### 取值

- [`<length>`](api/css/data-type/length.mdx)
  定义圆角的形状。

- [`<percentage>`](api/css/data-type/percentage.mdx)
  以百分比定义圆角的形状。
