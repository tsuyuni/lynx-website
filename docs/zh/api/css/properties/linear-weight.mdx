# linear-weight

## 介绍

子元素在[线性布局](guide/ui/layout/linear-layout.mdx)主轴方向上的权重。线性布局的权重概念类似于[弹性布局](guide/ui/layout/flexible-box-layout.mdx)的 [`flex-grow`](./flex-grow.mdx)。对于有设 `linear-weight` 子元素的线性容器：

- 首先会计算所有 `linear-weight: 0` 的子元素，并计算这些子元素占用主轴方向上的尺寸。
- 其余的 `linear-weight` 非 `0` 的子元素，将按照其 `linear-weight` 分配剩余的（线性容器减去所有的 `linear-weight: 0` 的子元素的主轴大小）主轴方向上的尺寸。具体来说，`linear-weight` 非 0 子元素尺寸 = 子元素对应的 `linear-weight` ÷ 所有子元素 `linear-weight` 之和 ✖️ 剩余的主轴方向上的尺寸

:::info
线性容器主轴大小必须是确定值（推荐，不然会有莫名其妙的表现）；

不支持同时使用固定值和 `linear-weight`，如果有 `linear-weight`，自动忽略固定值（性能优先）。
:::

## 语法

```css
/* <number> 值 */
linear-weight: 3;
linear-weight: 5;
```

### 取值

- [`<number>`](api/css/data-type/number.mdx)

  负值无效，**默认值**为 `0`。当[线性布局](guide/ui/layout/linear-layout.mdx)某子元素 `linear-weight` 取 `0` 时，该子元素的布局将不受 `linear-weight` 的影响。
