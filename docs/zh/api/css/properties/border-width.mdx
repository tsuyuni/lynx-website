# border-width

## 介绍

设置元件四个边框宽度的简写属性，对应 [`border-top-width`](./border-top-width.mdx)、[`border-right-width`](./border-right-width.mdx)、[`border-bottom-width`](./border-bottom-width.mdx)、[`border-left-width`](./border-left-width.mdx)。

## 语法

```css
border-width: thin medium thick 10px;
```

### 取值

- thin | medium | thick | [`<length>`](/api/css/data-type/length.html)
- 支持 1 - 4 值语法（上 → 右 → 下 → 左）

## 形式语法

```css
border-width = [ <line-width> ]{1,4}
<line-width> = <length> | thin | medium | thick

```

- [MDN 参考](https://developer.mozilla.org/zh-CN/docs/Web/CSS/border-width)
- 默认值不同（Web 默认值 `medium`）
- 不支持全局值
