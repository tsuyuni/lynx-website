# border-right

## 介绍

用于设置右边框相关样式，该属性可按顺序设置 `border-right-width`、`border-right-style`、`border-right-color`。

## 语法

```css
/* 右边框是粗双线红边框 */
border-right: thick double red;

/* 右边框是细点线蓝边框 */
border-right: thin dotted blue;

/* 右边框是10px虚线橙边框 */
border-right: 10px dashed orange;
```

### 取值

可按顺序设置如下属性：

- [`border-right-width`](./border-right-width.mdx)
  边框的宽度

- [`border-right-style`](./border-right-style.mdx)
  边框的样式

- [`border-right-color`](./border-right-color.mdx)
  边框的颜色

## 形式语法

```
<line-width> || <line-style> || <color>
where
<line-width> = <length> | thin | medium | thick
<line-style> = none | hidden | dotted | dashed | solid | double | groove | ridge | inset | outset
```

## 与 Web 的区别

- [mdn:border-right](https://developer.mozilla.org/en-US/docs/Web/CSS/border-right)

不支持 inherit、initial、revert、unset 等
