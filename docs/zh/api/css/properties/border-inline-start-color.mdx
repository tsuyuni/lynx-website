# border-inline-start-color

## 介绍

用于设置阅读模式开始方向的边框颜色。

在 `LTR` 模式下，`border-inline-start-color` 等价于 `border-left-color`。
在 `RTL` 模式下，`border-inline-start-color` 等价于 `border-right-color`。

## 语法

```css
border-inline-start-color: red;
border-inline-start-color: #ffbb00;
border-inline-start-color: rgb(255, 0, 0);
border-inline-start-color: hsla(100%, 50%, 25%, 0.75);
border-inline-start-color: currentColor;
border-inline-start-color: transparent;
```

### 取值

- `transparent`
  边框颜色为透明。

- [`<color>`](/api/css/data-type/color)
  边框具体的颜色。

- 默认值
  **block**
