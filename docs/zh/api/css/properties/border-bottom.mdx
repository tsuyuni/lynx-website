# border-bottom

## 介绍

用于设置下边框相关样式，该属性可按顺序设置 `border-bottom-width`、`border-bottom-style`、`border-bottom-color`。

## 语法

```css
/* 下边框是粗双线红边框 */
border-bottom: thick double red;

/* 下边框是细点线蓝边框 */
border-bottom: thin dotted blue;

/* 下边框是10px虚线橙边框 */
border-bottom: 10px dashed orange;
```

### 取值

按语法顺序接受以下值类型：

- [`<line-width>`](./border-bottom.mdx#形式定义)

  - 见 [`<border-bottom-width>`](./border-bottom-width.mdx)

- [`<line-style>`](./border-bottom.mdx#形式定义)

  - 见 [`<border-bottom-style>`](./border-bottom-style.mdx)

- [`<color>`](/api/css/data-type/color.html)
  - 见 [`<border-bottom-color>`](./border-bottom-color.mdx)

## 形式语法

```css

border-bottom =
  <line-width>  ||
  <line-style>  ||
  <color>

<line-width> = <length> | thin | medium | thick
<line-style> = none | hidden | dotted | dashed | solid | double | groove | ridge | inset | outset

```

## 与 Web 的区别

- [mdn:border-bottom](https://developer.mozilla.org/en-US/docs/Web/CSS/border-bottom)
- 不支持 inherit、initial、revert、unset 等全局值
