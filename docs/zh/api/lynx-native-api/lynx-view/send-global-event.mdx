# sendGlobalEvent

通过客户端向前端发送全局事件，前端可以通过 `GlobalEventEmitter` 监听该事件；

## 语法

### Android

```java
public void sendGlobalEvent(String name, JavaOnlyArray params);
```

### 参数说明

- `name: String`：全局事件名；
- `params: `: 全局事件携带的参数；

### iOS

```objc
- (void)sendGlobalEvent:(nonnull NSString*)name withParams:(nullable NSArray*)params;
```

### 参数说明

- `name: NSString*`: 全局事件名；
- `params: NSArray*`: 全局事件携带的参数；
