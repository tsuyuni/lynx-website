# removeLynxViewClient

与 `addLynxViewClient` 相对应，移除 `LynxView` 生命周期事件监听；

## 语法

### Android

```java
public void removeLynxViewClient(LynxViewClient client)
```

## 参数说明

- `client: LynxViewClient`: 客户端实现的结构，并注册给 `LynxView` 实例，用于获取 `LynxView` 生命周期各流程回调；
